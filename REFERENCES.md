### R Packages

Major R packages used in this analysis:

- `tidyverse` (Wickham et al.) for data manipulation and visualization
- `readxl` (Wickham and Bryan) for reading Excel files
- `here` (Müller) for project-relative file paths
- `janitor` (Firke) for data cleaning utilities
- `car` (Fox and Weisberg) for regression diagnostics including VIF and NCV tests
- `lmtest` (Zeileis and Hothorn) for the Breusch-Pagan test
- `sandwich` (Zeileis) for HC3 robust standard errors
- `broom` (Robinson, Hayes, and Couch) for tidying model output
- `modelsummary` (Arel-Bundock) for regression tables
- `corrplot` (Wei and Simko) for the correlation matrix figure
- `ggthemes` (Arnold) for plot styling

A complete list with version numbers is available in `output/session_info.txt`.
